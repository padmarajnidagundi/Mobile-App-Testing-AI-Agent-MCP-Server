## Overview

**The Useless Web** is a random-website generator and community platform at https://uselessweb.org/.  
Key features:
- One-click random redirect button ("🎲 Take me somewhere useless")
- Curated website cards with upvote system (trending + latest submissions)
- Category browsing, leaderboard, sign-up / community submission
- Stats counters (Websites, Members, Votes)
- Social links and FAQ section

**Test scope:**  
Full functional, edge-case, accessibility, performance, and mobile-specific coverage
across desktop web (Playwright), Android (Mobilewright), and iOS (Mobilewright).

## Accessibility Checks

### TC-A01 · Keyboard Navigation — Tab Reaches All Interactive Elements
Tab order must include: skip link (if present) → nav links → CTA → cards → footer links.

### TC-A02 · CTA Button Has Accessible Name
`role=button` or `<button>` element must have non-empty text content or `aria-label`.

### TC-A03 · Colour Contrast
All body text must meet WCAG 2.1 AA (contrast ratio ≥ 4.5:1 for normal text, ≥ 3:1 for large text).

### TC-A04 · Images Have Alt Text
Any decorative or informative `<img>` must have an appropriate `alt` attribute.

---

## Performance Checks

### TC-P01 · First Contentful Paint < 2.5 s
Measured via Lighthouse or Playwright `performance.getEntriesByType('paint')`.

### TC-P02 · Total Blocking Time < 300 ms
Long tasks should not block the main thread beyond 50 ms each.

### TC-P03 · No Console Errors on Load
Zero `console.error` messages on initial page load (excluding CORS pre-flight from 3rd parties).

---

## Mobile-Specific Checks

### TC-M01 · Portrait → Landscape Rotation
Rotate from portrait to landscape; content must reflow without horizontal overflow or broken layout.

### TC-M02 · CTA Button Touch Target Size ≥ 44 × 44 px
Measured via bounding box on mobile viewport (375×667 or 390×844).

### TC-M03 · `user-scalable=no` NOT Set
Meta viewport must not block pinch-to-zoom (`user-scalable=no` or `maximum-scale=1` are violations).

### TC-M04 · Swipe-to-Scroll Works
On Android/iOS, vertical swipe must scroll the page smoothly without horizontal drift.

### TC-M05 · Cards Tap-Friendly on Small Screens (320 px)
At 320 px width, website cards must not overlap and must be individually tappable.
